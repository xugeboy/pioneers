# Product Catalog Overrides (Pioneers)

This document defines page-specific layout rules for the product catalog.

## 1) Page Header
- **Title:** “Product Catalog”
- **Subcopy:** short statement about custom OEM/ODM capabilities.
- **Optional badge:** “B2B Ready / Factory Direct”.

## 2) Product Grid
- **Grid:** 2 columns (tablet), 3 columns (desktop), 1 column (mobile).
- **Card layout:** image (4:3) + model + title + 2–3 key specs.
- **Hover:** fade to secondary image (no scale).
- **Featured:** show “Featured” label for marked products.

## 3) Pagination
- **Placement:** bottom center only.
- **Style:** rounded buttons with clear border, strong hover contrast.
- **No sorting or filtering UI.**
