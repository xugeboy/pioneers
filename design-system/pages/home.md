# Home Page Overrides (Pioneers)

This document defines page-specific layout rules for the homepage. It overrides or expands the MASTER design system.

## 1) Hero Section
- **Layout:** vertical hero with left-aligned text + right industrial visual or product montage.
- **Content:** H1 + subcopy + two CTAs + certification badges.
- **CTA placement:** primary CTA above the fold; secondary CTA aligned beside.
- **Typography:** H1 uses Syncopate, uppercase, tracking +0.06em.

## 2) Horizontal Product Journey (Signature)
- **Purpose:** immersive product discovery.
- **Structure:**
  - Intro (short vertical section)
  - Horizontal track with scroll-snap cards
  - End CTA block with strong contrast
- **Interaction:** scroll-snap, progress indicator, and hover reveal.
- **Mobile:** fall back to stacked vertical cards with same content order.

### Track Card Design
- Card ratio: 4:3 image + 1:1 content.
- Hover state swaps primary/secondary image with opacity transition.
- Each card includes model code and use-case tag.

## 3) Bento Feature Grid
- **Layout:** 3–5 modular blocks.
- **Content:** manufacturing capability, QC tests, OEM/ODM support, lead time.
- **Emphasis:** one primary block with darker background.

## 4) Trust & Authority
- **Certificates row:** display ISO / testing badges (if available).
- **Client logos:** monochrome in one line or grid.
- **Metrics:** highlight sample time, MOQ, export coverage.

## 5) CTA Footer
- **Single goal:** “Request Quote”.
- **Design:** full-width block in CTA blue with white text.
